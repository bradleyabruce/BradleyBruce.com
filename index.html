    SpotCheck

    Project:
    SpotCheck is an end-to-end solution to provide mobile users the ability to quickly and reliably find parking wherever they are. Using machine learning to process images from cameras arrays stationed around local parking lots, our application suite determines where open parking is. This project combines machine learning, an ASP admin console application, and a Xamarin-built iOS and Android application. Using Google Maps API, users of the mobile app can get directions to parking lots with available parking spots. This spot availability is determined using pretrained TensorFlow data models of vehicles.
