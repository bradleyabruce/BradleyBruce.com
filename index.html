Projects

    Python Security Camera Server

    Project:
    My python security application is a two-part system made up of a single server machine running this server code, and any number of connected cameras (raspberry pis) running the client code (listed below). The client cameras each transmit a stream of images over a local network to the server machine. The server machine then scans the image stream to detect movement between frames and draws a rectangle around detected movement. Images are then stitched back into an mp4 file for long term storage for the user to watch later.

    Quarantine Calculation.com

    Project:
    Quarantine Calculation was a static website I created during the first round of economic shutdowns due to COVID-19 in early 2020. It uses javascript to generate graphs to estimate and illustrate how many people that someone with COVID-19 could infect if they did not follow quarantine guidelines. It is hosted on Amazon Web Services with a domain purchased through Google Domains. It was based on preliminary that had been collected at the time of the website creation and should not be used in an official capacity to estimate COVID-19 infections.

    SpotCheck

    Project:
    SpotCheck was created as my senior capstone project by myself and two other students. SpotCheck is an end-to-end solution to provide mobile users the ability to quickly and reliably find parking wherever they are. Using machine learning to process images from cameras arrays stationed around local parking lots, our application suite determines where open parking is. This project is made up of a python machine learning application, an ASP.NET admin console application, and a Xamarin-built iOS and Android application. Using Google Maps API, users of the mobile app can get directions to parking lots with available parking spots. This spot availability is determined using a pretrained TensorFlow data model.
